    Apollo Control Board Overview

    The Prometheus MSLA controller board (Apollo) is the official controller/motherboard for the Prometheus MSLA printer.
    The motherboard is fully open source (under the license CERN-OHL-S-2.0) so it can be used in other similar machines and more.

    All documentation for it can be found on the Apollo Github
